### Prueba de conocimientos de PHP,Laravel, MySQL, HTML y JavaScript

**Problema**
**Crear sistema web para creación de post en un blog**

En el sistema los usuarios pueden crear una cuenta, iniciar sesión, recuperar contraseña y crear/editar/eliminar post del blog.
El sistema debe permitir consumir mediante una [API REST](https://asiermarques.com/2013/conceptos-sobre-apis-rest/) los blogs a una aplicación de un tercero, para esto la aplicación del tercero necesita conocer el listado de los blogs ordenado por el más reciente y acceder al detalle de un blog en formato JSON


Para que un usuario se pueda registrar en la plataforma necesita:
- Nombre *
- Email *
- Descripción de perfil
- Contraseña *

Cada blog necesita:
- Título del blog*
- Dos o más imagenes para portada
- Descripción del blog*

>  Datos obligatorios *
